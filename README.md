# ACR Command line tool


```bash
acr:
  acr operates against an OCI conformant registry

Usage:
  acr [options] [command]

Options:
  --registry <registry>    Registry Login Server [default: sajay.azurecr.io]
  --username <username>    Registry Username [default: sajay]
  --password <password>    Registry Login Server [default: *****]
  --verbose                Enable verbose logging [default: False]
  --version                Show version information
  -?, -h, --help           Show help and usage information

Commands:
  repository          Repository operations
  manifest            Manifest operations
  tag                 Tag operations
  pull <reference>    Pull Artifact
  layer               Layer Operations
  config              Manifest operations
```
